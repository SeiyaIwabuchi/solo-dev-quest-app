# 個人開発支援アプリ 企画書

## 📱 アプリ概要

個人開発者が抱える「モチベーション維持」「孤独感」「知識不足」の3大課題を解決する、AI×ゲーミフィケーションを活用した開発支援プラットフォーム。

**ターゲット**: 個人でアプリ・Webサービス開発を行う20-40代の開発者

---

## 🎯 解決する3大課題

### 1. モチベーション維持の難しさ
**課題**: 長期的な開発を一人で続けるのは精神的に辛い

**解決策**:
- 🏃 **先進的な進捗可視化**: タスク完了でマラソンランナーがゴールに近づく
  - 初期: マラソンランナー
  - 今後追加: パズル、都市建設、植物育成、宇宙探索、ダンジョン攻略、RPGキャラ成長など
  - ユーザーが好きなテーマを選択可能
- 🤖 **AIによる褒めシステム**: タスク完了時にAIが具体的に褒めてくれる
- 😤 **AIによる説教システム**: サボると段階的にAIが叱ってくれる（軽い注意→厳しい叱責→失望）
- 🎮 **ゲーミフィケーション**: アプリ内通貨（DevCoin）を貯めて使える

### 2. 開発の孤独感
**課題**: 一人で作業していると誰とも話せず孤立する

**解決策**:
- 🤝 **AI仮想クライアント**: AIがクライアント役になって定期ミーティングや相談相手に
- 📱 **SNS連携**: X、Threads、Instagramに進捗を簡単投稿
  - 進捗可視化の成長動画を自動生成・投稿
- 🌐 **ハッシュタグタイムライン**: 特定ハッシュタグ（例: #個人開発チャレンジ）の投稿をアプリ内に収集
  - フォロー関係なく全ユーザーの投稿が見られる
  - アプリ内からいいね・コメント可能

### 3. 技術的な知識不足
**課題**: わからないことがあっても気軽に聞ける相手がいない

**解決策**:
- 💰 **Q&Aプラットフォーム**: 
  - 質問投稿にDevCoinを消費（質の高い質問を促進）
  - 回答者はDevCoinを獲得（知識共有のインセンティブ）
  - 課金でDevCoinを購入可能
- 📚 **ナレッジベース**: 過去のQ&Aを検索・参照できる

---

## 💰 アプリ内経済システム「DevCoin」

### 無料DevCoin
- **獲得方法**:
  - ✅ タスク完了・継続ボーナス
  - 💬 質問への回答提供
  - 👍 コミュニティ貢献（いいね獲得など）
- **有効期限**: なし
- **法的位置づけ**: 対価を得ていないため資金決済法の規制対象外

### 有料DevCoin（課金）
- **購入**: 100円 = 100 DevCoin
- **有効期限**: 6ヶ月（資金決済法の規制回避）
- **消費優先順位**: 有料DevCoinから優先的に消費

### 使い道
- ❓ 技術的質問の投稿（10 DevCoin/回）
- 💬 AI相談チャット（10 DevCoin/回）
- 🤖 AI仮想クライアントミーティング（50 DevCoin/回）
- 🛡️ AI説教の回避（20 DevCoin）
- 🎁 プレミアム可視化テーマの解放

### 利用規約
- 払い戻し不可
- 換金不可
- 有料DevCoinは購入から6ヶ月で失効
- 無料DevCoinと有料DevCoinは分けて管理

## 🎨 進捗可視化システムの詳細

### マラソンランナー（初期実装）
- プロジェクトの完了度に応じてランナーがゴールに向かって走る
- タスク完了で距離が進む
- 天候システム: 順調なら晴天、遅れると雨
- ランナーの見た目をDevCoinでカスタマイズ可能

### 今後追加予定のテーマ
- **パズル完成**: タスクがピース、完成で美しいアート
- **3D都市建設**: タスク達成で建物が建つ
- **植物育成**: 継続で成長、サボると枯れる
- **宇宙探索**: タスク完了で新惑星発見
- **ダンジョン攻略**: 階層を進み、ボス戦で大きな達成感
- **RPG成長**: スキル習得で新能力解放

ユーザーが設定画面で好きなテーマを選択可能

## 🤖 AI機能の詳細

### 褒めシステム
- タスク完了時に自動で励まし
- ユーザーの成長パターンに合わせたメッセージ
- 具体的な成果を言及

### 説教システム
- サボり検知: 未完了タスク、長期間未ログインなど
- 段階的エスカレート:
  1. 軽い注意「最近開発してないけど大丈夫？」
  2. 厳しい叱責「このままだとプロジェクト終わらないよ」
  3. 失望表現「...頑張ると言ってたのに」
- キャラクター性格設定: 厳格師匠、心配する親など
- DevCoinで説教回避も可能

### 仮想クライアント
- 定期的なミーティング機能
- プロジェクト相談・壁打ち相手
- クライアントの性格・業界をカスタマイズ可能

## 🌐 SNS連携機能

### 外部投稿
- X（Twitter）、Threads、Instagramに直接投稿
- 進捗可視化の成長動画を自動生成
- 投稿テンプレートで簡単共有

### ハッシュタグタイムライン
- 特定ハッシュタグ付き投稿を自動収集
- フォロー関係なく表示（公開アカウントのみ）
- アプリ内からいいね・コメント可能
- ※技術的実装可能性要検証（PoC必須）

## 💻 技術構成

### フロントエンド
- **Flutter** (iOS/Android/Web対応)
- **状態管理**: Riverpod
- **認証**: Firebase Auth
- **プッシュ通知**: Firebase Cloud Messaging

### バックエンド
- **認証**: Firebase Authentication
- **データベース**: Cloud Firestore
- **ストレージ**: Firebase Storage
- **サーバーレス関数**: Cloud Functions (Node.js/TypeScript)
- **ホスティング**: Firebase Hosting (Web版)
- **リアルタイム通信**: Firestore Realtime Listeners

### AI機能
- **メインAI**: Claude 3.5 Haiku (Anthropic)
  - 高速・低コスト
  - 日本語対応が優秀
  - コンテキスト理解が自然
- **フォールバックAI**: GPT-4o mini (OpenAI)
  - メインAI障害時の代替
  - API制限時の切り替え
- **実行環境**: Cloud Functions

### インフラ
- **開発**: Firebase Emulator Suite
- **本番**: Firebase (Spark/Blaze Plan)
- **CI/CD**: GitHub Actions
- **コスト**: 初期は無料枠、スケール後も月数千円程度

## 💵 マネタイズ戦略

### 1. 有料DevCoin販売
- **価格**: 100円 = 100 DevCoin、500円 = 550 DevCoin（10%ボーナス）等
- **有効期限**: 6ヶ月（資金決済法対応）
- **用途**: AI機能、質問投稿、プレミアムテーマ解放

### 2. プレミアムプラン（月額680円）
- すべてのAI機能無制限
- 広告なし
- 毎月200 DevCoin（無料）ボーナス
- 限定可視化テーマ

### 3. スーパープレミアムプラン（月額1,480円）
- プレミアム特典すべて
- AI仮想クライアント機能
- 優先サポート
- 毎月500 DevCoin（無料）ボーナス

### 4. 広告収入
- 無料ユーザー向けの技術系企業・サービス広告

### 法的対応
- 利用規約に明記（払い戻し不可、換金不可、有効期限）
- 事業者情報・苦情相談窓口の表示
- 無料DevCoinと有料DevCoinの分離管理

## 💡 このアプリの独自性

### 既存の目標管理アプリとの違い
❌ 既存アプリ: シンプルなチェックリストやグラフ表示だけ
✅ このアプリ: ゲーム的な進捗可視化 + AI伴走 + コミュニティ

### 既存のSNSとの違い
❌ 既存SNS: 雑多な情報に埋もれる、フォロワーが必要
✅ このアプリ: 個人開発者だけの空間、ハッシュタグで自動集約

### 既存のQ&Aサイトとの違い
❌ 既存サイト: 敷居が高い、回答がつかないことも
✅ このアプリ: 通貨による質問・回答のインセンティブ設計
