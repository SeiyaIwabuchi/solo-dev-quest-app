# 個人開発支援アプリ 要件定義書

## アプリ概要
個人開発者の3大課題「モチベーション維持」「孤独感」「知識不足」を解決するオールインワン支援アプリ

## 解決したい課題
1. **モチベーション維持** - 長期的な開発継続が困難
2. **孤独感** - 一人で作業することの心理的負担
3. **知識不足** - 技術的な壁に直面した際の解決手段不足

## 主要機能

### 1. モチベーション維持機能
- **カスタマイズ可能進捗可視化システム**
  - **初期実装**: マラソンランナーがゴールに向かって走る（完了度に応じて距離を進む）
  - **将来追加予定**: 
    - パズル完成（タスクがピース、完成で美しいアート作品）
    - 3D都市建設（タスク達成で建物建設）
    - 植物育成（継続で成長、サボると枯れる）
    - 宇宙探索（タスク完了で惑星発見）
    - ダンジョン攻略（タスク完了で階層を進み、ボス戦で大きな成果を実感）
    - RPG風キャラクター成長
    - その他ユーザーリクエストに応じて追加
  - **選択システム**: ユーザーが好きなテーマを設定画面で選択可能
  - **実装戦略**: アップデートで段階的にテーマ追加、コミュニティ投票で次回追加テーマを決定

- **AI褒めシステム**
  - タスク完了時の自動称賛
  - 継続日数に応じた励ましメッセージ
  - 個人の成長パターンに合わせたフィードバック

- **ゲーミフィケーション**
  - アプリ内通貨（DevCoin等）の獲得システム
  - タスク達成・継続ボーナス
  - レベルアップ・ランキング機能

- **AI説教システム**
  - サボり検知（未完了タスク、長期間未ログイン等）
  - 段階的エスカレート（注意→叱責→失望）
  - キャラクター性格設定（厳格師匠、心配親等）
  - 説教回避機能（通貨消費、正当理由申告）

### 2. 孤独感軽減機能
- **AI仮想クライアント**
  - 定期ミーティング機能
  - プロジェクト相談・壁打ち相手
  - クライアントキャラクターのカスタマイズ

- **ハッシュタグベース交流機能**
  - **外部投稿**: X（Twitter）、Threads、Instagramへの直接投稿
  - **進捗動画共有**: 進捗可視化システム（マラソンランナー等）の0→1成長を動画で自動生成・投稿
  - **投稿テンプレート**: 「今日は○○のタスクを完了！ランナーが△△km進みました🏃‍♂️ #個人開発チャレンジ」等
  - **ハッシュタグタイムライン**: 
    - 特定ハッシュタグ（例：#個人開発チャレンジ）付き投稿をアプリ内に自動収集
    - フォロー関係に関係なく表示（公開アカウントのみ）
    - 鍵垢・非公開投稿は表示対象外
  - **アプリ内インタラクション**:
    - アプリ内からいいね・リツイート・コメント可能
    - 各SNSのAPI経由で実際のアクション実行
    - ※技術的実装可能性要検証（PoC必須）

- **コメント・応援機能**
  - テンプレート化されたコメント入力
  - いいね・応援ボタン
  - 相互フォロー機能

### 3. 知識不足解決機能
- **Q&Aプラットフォーム**
  - 技術的質問の投稿（通貨消費）
  - 回答者への報酬システム（通貨獲得）
  - 質問・回答の評価機能

- **ナレッジベース**
  - よくある質問のデータベース
  - 過去の質問・回答の検索
  - 分野別カテゴリ分類

- **学習支援**
  - AIによる技術的アドバイス
  - 参考資料・ドキュメントの提案
  - スキルレベル別学習パス提案

## アプリ内経済システム

### 無料DevCoin
- **獲得方法**:
  - タスク完了・継続ボーナス
  - 質問への回答提供
  - コミュニティ貢献（いいね獲得等）
- **有効期限**: なし
- **法的位置づけ**: 対価を得ていないため資金決済法の規制対象外

### 有料DevCoin
- **購入価格**: 100円 = 100 DevCoin、500円 = 550 DevCoin（10%ボーナス）等
- **有効期限**: 購入から6ヶ月（資金決済法対応）
- **消費優先順位**: 有料DevCoinから優先的に消費

### 消費方法
- 技術的質問の投稿（10 DevCoin）
- AI相談チャット（10 DevCoin/回）
- AI仮想クライアントミーティング（50 DevCoin/回）
- AI説教回避（20 DevCoin）
- プレミアム可視化テーマ解放（各50-100 DevCoin）

### 利用規約での明記事項
- 払い戻し不可
- 換金不可
- 有料DevCoinは購入から6ヶ月で失効
- 無料DevCoinと有料DevCoinの分離管理
- 事業者情報・苦情相談窓口の表示

## マネタイズ戦略

### 1. 有料DevCoin販売
- 100円 = 100 DevCoin
- 500円 = 550 DevCoin（10%ボーナス）
- 1,000円 = 1,150 DevCoin（15%ボーナス）
- **有効期限**: 6ヶ月（資金決済法対応）

### 2. プレミアムプラン（月額680円）
- すべてのAI機能無制限
- 広告なし
- 毎月200 DevCoin（無料扱い）ボーナス
- 限定可視化テーマ

### 3. スーパープレミアムプラン（月額1,480円）
- プレミアム特典すべて
- AI仮想クライアント機能
- 優先サポート
- 毎月500 DevCoin（無料扱い）ボーナス

### 4. 広告収入
- 技術系企業・サービスの広告掲載（無料ユーザー向け）

### 法的対応事項
- 利用規約への必須記載（払い戻し不可、換金不可、有効期限）
- 事業者情報・苦情相談窓口の表示
- 無料/有料DevCoinの分離管理システム
- 有効期限管理・通知機能

## ターゲットユーザー
- **プライマリ**: 個人でアプリ・Webサービス開発を行う人
- **セカンダリ**: フリーランス開発者、副業開発者
- **年齢層**: 20-40代
- **スキルレベル**: 初級者から上級者まで

## 技術的考慮事項

### フロントエンド
- **Flutter 3.16+** (iOS/Android/Web)
- **状態管理**: Riverpod
- **Firebase連携**: Firebase Core, Auth, Firestore, Storage, Messaging, Analytics

### バックエンド（Firebase構成）
- **認証**: Firebase Authentication
- **データベース**: Cloud Firestore
- **ストレージ**: Firebase Storage
- **サーバーレス関数**: Cloud Functions (Node.js/TypeScript)
- **ホスティング**: Firebase Hosting (Web版)
- **リアルタイム通信**: Firestore Realtime Listeners

### 開発環境
- **ローカル開発**: Firebase Emulator Suite
  - Authentication Emulator
  - Firestore Emulator
  - Functions Emulator
  - Storage Emulator
- **メリット**:
  - 本番環境に影響を与えずに開発
  - 無料で無制限にテスト可能
  - オフライン開発可能
  - データのリセット・初期化が容易
- **環境分離**: 
  - 開発: Firebase Emulator（ローカル）
  - ステージング: Firebase プロジェクト（Spark Plan）
  - 本番: Firebase プロジェクト（Blaze Plan）

### AI機能
- **メインAI**: Claude 3.5 Haiku (Anthropic)
  - 高速・低コスト（褒め約0.028円/回）
  - 日本語の自然さに優れる
- **フォールバックAI**: GPT-4o mini (OpenAI)
  - メイン障害時・API制限時の代替

### インフラ
- **開発**: Firebase Emulator Suite（完全ローカル）
- **ステージング/本番**: Firebase Spark/Blaze Plan
- **CI/CD**: GitHub Actions
- **コスト見積もり**: MAU 1,000人で月3,833円（AI使用料）

## 成功指標（KPI）
- **継続率**: 月間アクティブユーザー維持率
- **エンゲージメント**: 1日平均利用時間
- **コミュニティ活性度**: 質問・回答・コメント数
- **収益**: 課金ユーザー数・平均課金額

## 開発優先順位

### Phase 0: 開発環境構築・超軽量MVP (1ヶ月)

**開発環境セットアップ:**
- Firebaseプロジェクト作成（開発/ステージング/本番）
- Firebase Emulator Suite インストール・設定
  ```bash
  firebase init emulators
  # Auth, Firestore, Functions, Storage を選択
  ```
- Flutterプロジェクト初期化
- Firebase Emulatorへの接続設定
  ```dart
  // 開発時のみEmulator使用
  if (kDebugMode) {
    await FirebaseAuth.instance.useAuthEmulator('localhost', 9099);
    FirebaseFirestore.instance.useFirestoreEmulator('localhost', 8080);
    await FirebaseStorage.instance.useStorageEmulator('localhost', 9199);
  }
  ```

**MVP機能実装:**
- ユーザー認証（メール/パスワード）
- 基本的なタスク管理（プロジェクト・タスクCRUD）
- マラソンランナー可視化（シンプル版）
- AI褒めシステム（Claude Haiku、1日3回制限）
- DevCoin獲得機能（無料のみ）

**開発フロー:**
1. Firebase Emulatorでローカル開発
2. 機能完成後、ステージング環境でテスト
3. 問題なければ本番デプロイ

### Phase 1: コア機能拡張 (1-2ヶ月)
- 有料DevCoin購入機能
- DevCoin消費機能（質問投稿等）
- 外部SNS投稿機能（X/Threads/Instagram）
- AI相談チャット（基本版）
- プッシュ通知・リマインダー
- 利用規約・プライバシーポリシー実装

### Phase 2: コミュニティ機能 (1-2ヶ月)
- Q&Aプラットフォーム（質問・回答・評価）
- ハッシュタグタイムライン（PoC検証後）
- アプリ内いいね・コメント機能
- プレミアムプラン導入

### Phase 3: 高度なAI機能 (2-3ヶ月)
- AI仮想クライアント
- AI説教システム（段階的エスカレート）
- AIキャラクター性格カスタマイズ
- 追加可視化テーマ（パズル、都市建設等）

### Phase 4: スケーリング・最適化 (継続的)
- パフォーマンス最適化
- ユーザーフィードバック対応
- 新テーマ・新機能追加
- スーパープレミアムプラン導入
- 分析・統計機能
